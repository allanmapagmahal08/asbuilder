# Allan Simbulan Builder — Portfolio

Personal-brand portfolio for **Allan Simbulan**, AI Automation & GoHighLevel Specialist (Manila, Philippines).

- **Live:** https://asbuilder.ph
- **Stack:** single static page (`ai-automation.html`) — Tailwind (Play CDN) + Phosphor icons, no build step.
- **Features:** dark/light themes, animated starfield background, custom cursor, project case-study modals, ROI calculator, live GoHighLevel booking calendar.

The root (`/`) serves the portfolio via the rewrite in `vercel.json`.
